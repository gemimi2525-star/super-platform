# System Design: macOS-grade UI Foundation

## Goals
To create a premium, responsive, and intuitive interface inspired by macOS design principles.

## Design Tokens
- Typography: SF Pro / Inter
- Spacing: 4px grid
- Radius: 8px / 12px
- Shadows: Multi-layer soft shadows

## Core Layout Patterns
- AppShell: Sidebar + Topbar + Content
- Glassmorphism effects

## Component Inventory
- Button
- Card
- Input
- Modal
