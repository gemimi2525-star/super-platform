Usage Guidance

Best practices for typography in enterprise SaaS interfaces.

‚úì Do
  • Use --color-text-primary for body text (neutral-900)
  • Use --color-text-secondary for hints and metadata (neutral-600)
  • Use --line-height-relaxed for long-form content (1.625)
  • Use negative letter-spacing only for large headings (H1-H3)
‚úó Don't
  • Don't use bold weight excessively‚Äîmedium (500) is often enough
  • Don't use pure black (#000) for text‚Äîuse neutral-900 for softer contrast
  • Don't use tight line-height for body text‚Äîreduces readability
  • Don't mix more than 2-3 font weights in a single view
